# Documentación de la Práctica 2

Esta carpeta contiene el documento final solicitado para la Práctica 2.

## Archivos

* `PRACTICA2.pdf`: archivo final en PDF para entrega.
* `PRACTICA2.docx`: versión editable del documento.

## Estructura del documento

El documento contiene portada, introducción, desarrollo del modelo estrella, tablas de dimensiones, tabla de hechos, proceso ETL, resultados, conclusión, enlace al repositorio y bibliografía en formato APA.
